listenOnce

  • listenOnce(target: EventTargetLike, type: string, listener: ListenerFunction, opt_this?: undefined | object): EventsKey
    • Defined in Core/events.ts:199

    Registers a one-off event listener on an event target. Inspired by https://google.github.io/closure-library/api/source/closure/goog/events/events.js.src.html

    This function efficiently binds a listener as self-unregistering listener to a this object, and returns a key for use with {@link module:ol/events~unlistenByKey} in case the listener needs to be unregistered before it is called.

    When {@link module:ol/events~listen} is called with the same arguments after this function, the self-unregistering listener will be turned into a permanent listener.

    Parameters

    • target: EventTargetLike

      Event target.

    • type: string

      Event type.

    • listener: ListenerFunction

      Listener.

    • Optional opt_this: undefined | object

      Object referenced by the this keyword in the listener. Default is the target.

    Returns EventsKey

    Key for unlistenByKey.
